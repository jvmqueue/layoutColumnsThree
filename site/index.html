Equal height columns

It does not matter how much content is in each column, the background colours will always stretch down to the height of the tallest column.

3 Column Dimensions

Each column is 33.33% percent wide with 2 percent padding on each side.

No CSS hacks

The CSS used for this 3 column layout is 100% valid and hack free. To overcome Internet Explorer's broken box model, no horizontal padding or margins are used. Instead, this design uses percentage widths and clever relative positioning.

No JavaScript

JavaScript is not required. Some website layouts rely on JavaScript hacks to resize divs and force elements into place but you won't see any of that nonsense here.

No Images

This three column layout requires no images. Many CSS website designs need images to colour in the column backgrounds but that is not necessary with this design. Why waste bandwidth and precious HTTP requests when you can do everything in pure CSS and HTML?

Valid XHTML strict markup

The HTML in this layout validates as XHTML 1.0 strict.

This layout is FREE for anyone to use

You don't have to pay anything. Simply view the source of this page and save the HTML onto your computer. My only suggestion is to put the CSS into a separate file. If you are feeling generous however, link back to this page so other people can find and use this layout too.
